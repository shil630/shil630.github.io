接触德州扑克没多久，我连续赢了几场，我开始觉得自己可能有点天赋。

手牌范围宽，没中牌也敢下重注，运气好点中了牌就更无所畏惧。因为我持续施压，经常没到摊牌就赢了。我拿着没什么胜率的牌，也能把筹码收到面前。

直到有经验的玩家开始“惩罚”我，当他们开始更高频率抓我的诈唬，我的打法很快就出了问题。没中牌时我还是习惯大尺度下注，中了牌时又容易把底池搞得失控。过去别人弃牌，我以为是自己打得好。后来别人跟注，我才发现不少下注是摊不开牌。

靠运气和胆量拿到的收益，很快又还了回去。

好结果奖励了我的胆量，也暂时遮住了我的漏洞。

> **上风期没有让我变强，只是让我的漏洞暂时看起来像风格。**

## 同一堂课，我在特斯拉上又交了一次学费

不过这堂课我并没有一次学会。后来在投资里，我又用另一种方式经历了一遍。

2024年初我开始建仓特斯拉。市场恐慌下跌时，我持续买入。后来特斯拉一路反弹，在2025年末创出新高。账面浮盈相当可观。

我把这段结果理解成对自己的肯定：左侧买入、下跌中持续加仓，看来是对的。于是反弹到高位我又增加了不少仓位。成本被一点点抬高，但看着不断增长的浮盈，这个变化并不显得危险——更像是在为一个已经被市场证明的判断加码。

进入2026年后，特斯拉重新走弱，浮盈大幅回撤。

这里最容易得出的结论是：高位加仓错了。但如果我只因为后来跌了，就把前面的动作全部判错，其实还是在用结果替代判断。

我真正需要复盘的不是“高位能不能买”，而是另一个问题：

> **如果没有前面那段上涨，我还会不会用同样的理由、同样的尺度，在那个位置继续增加仓位？**

这个问题让我有些不舒服。因为推动我加仓的证据里，不只有对公司的长期判断，还有前面那段丰厚浮盈带来的自信。股价上涨不只增加了账户余额，也悄悄增加了我对自己的评价。

## 好结果为什么更容易教错人

德扑和特斯拉，表面看是两种不同的错误。但它们的共同点是：我先得到了好结果，然后把更多收益归因给自己，更相信原来的做法，扩大参与范围——直到环境改变，才看见原来的脆弱性。

亏钱时，人很容易想到复盘。赢钱时，我们更想总结成功经验。但一次盈利里混合了太多东西：自己的判断、市场的环境、无法控制的随机结果。它们不会在账单上分别列出来。

真正危险的不是亏钱，而是结果不错、过程却很脆弱。

牌桌上，别人连续弃牌，让我以为自己的范围和尺度没有问题。投资里，股价持续上涨，让我把“这次赚到了”慢慢翻译成“这套做法已经被证明了”。

好结果不只是让人开心。它还会让人悄悄扩大原来的动作：

```text
- 手牌范围变得更宽
- 下注尺度变得更重
- 买入频率变得更高
- 仓位边界变得更松
- 原本应该补充的证据，被账户里的盈利替代

```

*图注：两段经历共享的不是同一种策略，而是同一种放大路径：好结果抬高自我评价，随后扩大范围、尺度、频率或仓位。*

## 好结果和好过程，是两张成绩单

我试着把一笔决定拆成两张成绩单：

|  | **结果不错** | **结果不好** |
| :--- | :--- | :--- |
| **过程清楚** | 值得复用，但仍要检查环境和运气 | 可能是正常波动，先看证据有没有变化 |
| **过程脆弱** | 最容易误学，优先检查自己放大了什么 | 问题比较明显，先停止继续放大 |

![二乘二矩阵：横向比较结果不错与结果不好，纵向比较过程清楚与过程脆弱；其中结果不错但过程脆弱是最容易误学的象限，应优先检查盈利后放大了什么。](/assets/images/posts/winning-streak-illusion/result-process-matrix-blog.png)

*图注：结果和过程要分开评分；最容易误学的，是结果不错掩盖了脆弱过程。*

坏结果加上清楚过程，虽然难受，却还能复盘；坏结果加上脆弱过程，问题更是一眼可见。

真正危险的是**结果不错、过程却很脆弱**。

要判断一笔盈利到底是能力还是运气，不能只看一次结果，而要问自己几个问题：

### 1. 结果出现前，我留下了什么

如果一笔交易上涨以后，我才能完整讲出买入理由，那些理由里可能已经混入了结果。

事前记录不需要很长。至少要让我看见：当时为什么参与、为什么是这个尺度、什么证据会改变判断。

### 2. 同一个环境里，别人发生了什么

如果大盘、行业和同类资产都在上涨，我当然可以为自己的选择高兴，但不能把全部结果都记进个人能力。

这不是为了否定自己，而是避免把环境顺风当成可以无限放大仓位的许可证。

### 3. 盈利以后，我扩大了什么

这是我以前最少问的一句。

上涨后增加仓位，究竟是因为出现了新的公司证据，还是因为账面盈利让我更确定？诈唬成功后继续重注，究竟是对手和牌面适合，还是因为上一次没人跟？

结果本身也可以成为信息，但它不能替代其他证据。

### 4. 换一个环境，这套过程还剩下什么

真正可重复的能力，不是机械复制同一个动作。

德扑里对手变了，范围和尺度本来就应该调整；投资里估值、公司和市场环境变化，原来的判断也必须更新。

可重复的是那套过程：先看证据，再决定是否参与；先写边界，再决定投入多少；环境变化后，知道该复核、收缩还是退出。

## 盈利以后，给自己补一张复盘卡

如果你最近刚好有一笔不错的盈利，不必急着否定它，也不用故意把功劳全部推给运气。

可以等庆祝完以后，补一张“上风期复盘卡”：

| 字段 | 要回答的问题 |
| :--- | :--- |
| 当时决定 | 结果出现前，我实际写下或说得出的判断是什么？ |
| 外部顺风 | 大盘、行业、对手或随机结果给了我什么帮助？ |
| 自己做对 | 哪个动作有事前依据，而且我仍愿意重复？ |
| 被错误奖励 | 哪个脆弱动作这次没有付出代价？ |
| 放大动作 | 盈利以后，我扩大了范围、尺度、仓位还是频率？ |
| 重来一次 | 换一个环境，我会保留什么、删除什么？ |

不用复盘整个账户。先选最近三笔盈利，逐笔写下来。五分钟就够了。

## 我后来先收紧的，不是胆量，是范围

德扑输钱以后，我开始系统学习。先强化翻前范围，不是每一手看起来“也能玩”的牌都要参与。然后调整下注尺度，不再把重注当成主动的唯一表达，同时提高弃牌率。

以前我会觉得，敢打、敢跟、敢把筹码推进去才算有判断。后来才慢慢接受：主动不是每一手都进攻，弃牌本身也是长期策略的一部分。

投资上，我还不能说自己已经完成了同样的改造。特斯拉这段经历也没有因为浮盈回撤，就自动给我一个清晰答案。公司的长期判断是否成立、仓位是否仍然合适，还是要回到证据和边界里继续检查。

但我想先给自己的加仓动作补三句：

```text
这次增加仓位，出现了什么新证据？
如果没有前面的浮盈，我还会用同样的尺度吗？
增加以后，整笔持仓最坏会让我失去多少主动权？
```

这三句不会替我判断特斯拉，也不能保证下一次不犯错。它们只是把“我更有信心了”，重新拆成可以检查的理由和尺度。

亏损会提醒我们哪里疼。盈利更值得多问一句：这次结果到底在训练我什么？

上风期当然值得高兴。只是别急着把每一笔盈利都刻进自己的能力证明里。先留下那些换一手牌、换一个行情，也愿意继续重复的过程。
