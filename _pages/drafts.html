草稿箱

未发布的文章，仅供本地预览

草稿存放在 _drafts/ 目录，部署到 GitHub Pages 时不会发布。

本地预览请运行：

bundle exec jekyll serve --draft

正式发布时，将文件移至 _posts/，并重命名为 YYYY-MM-DD-标题.md。

{% assign drafts = site.posts | where_exp: "post", "post.draft == true" %} {% if drafts.size > 0 %}
    {% for post in drafts %}
  • {{ post.date | date: "%Y-%m-%d" }} 草稿 {% if post.lang == 'en' %}EN{% else %}中文{% endif %}

    {{ post.title }}

    {% if post.subtitle %}

    {{ post.subtitle }}

    {% endif %}

    {{ post.excerpt | strip_html | truncate: 140 }}

    {% if post.tags.size > 0 %}
    {% for tag in post.tags %}#{{ tag }}{% endfor %}
    {% endif %}
    • {% endfor %}
{% else %}

当前看不到草稿。请用 --draft 参数启动本地服务器。

{% endif %}
